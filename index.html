Core Principles

We follow these principles in everything we do.

Keep it Simple

Using a software shouldn't require a PhD. It should be simple and easy to use and solve a really pain. This is easy to say than do. We fight with this every day.

No half-baked products

We believe in quality product. We prefer to release less but quality features rather than many but half-backed features.

Speed

Software should perform fast and provide great user experience.

Stick to Basics

We try to avoid getting into the pitfalls of conforming to the latest trends and stick to basics.
